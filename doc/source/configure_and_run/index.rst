Configure and run Solum
=======================

Configuration Reference
-----------------------

To alter the default compute flavor edit /etc/solum/templates/*.yaml
::

  flavor:
    type: string
    description: Flavor to use for servers
    default: m1.tiny

Edit the default section to the desired value.
